Создание сущностей в Системе (Версия 1.1)

Класс

Обязательно для заполнения:
  • имя Класса
  • один атрибут
Необязательно для заполнения:
  • описание Класса
  • больше одного атрибута

Атрибут

Обязательно для заполнения:
  • имя Атрибута
  • тип Атрибут
  • длина Атрибута (для STRING)
  • тип контента Атрибута (для MMEDIA)

Ресурс

Обязательно для заполнения:
  • идентификатор Класса
  • имя Ресурса
  • тип контента Ресурса

Объект

Обязательно для заполнения:
  • идентификатор Класса
  • Атрибуты Класса, указанные как "NOT NULL"
Необязательно для заполнения:
  • Атрибуты Класса, без указания "NOT NULL"

Отношения

  • Ассоциации между Классами

    Обязательно для заполнения:
    • имя Ассоциации
    • идентификатор Класса - источника
    • идентификатор Класса - цели

    Ассоциации между Объектами

    Обязательно для заполнения:
    • идентификатор Ассоциации между Классами
    • идентификатор Объекта - источника, который реализует Класс - источник
    • идентификатор Объекта - цели, который реализует Класс - цель


  • Агрегации между Классами

    Обязательно для заполнения:
    • имя Агрегации
    • идентификатор Класса - источника
    • идентификатор Класса - цели

    Агрегации между Объектами

    Создаются системой автоматически при обновлении или сохранении поля типа XMEMO.

Шаблон

Шаблон - это XML текст.
Шаблон может содержать:
  • HTML разметку (важно! необходимо использовать всегда парные метки, например <br></br> вместо <br>)
  • теги Системы для Шаблона:
    • Вставить данные из конкретного поля Объекта
      <field>fieldName</field>

    • Вставить результаты конкретного Шаблона (id = 5) каждого Объекта, имеющего Экземпляр Ассоциации конкретной Ассоциации (id = 2).
      <association associationId="2" templateId="5" type="object"></association>

    • Вставить гиперссылку на конкретный Шаблон (id = 5) каждого Объекта, имеющего Экземпляр Ассоциации конкретной Ассоциации (id = 2) и использовать сформировать строку из значений полей .filed1 … filedN, которая станет названием гиперссылки.
      <association associationId="2" templateId="5" type="hyperlink" >filed1 … filedN</association>

    • Вставить Ресурс(id = 2) принадлежащий Классу этого Шаблона.
      <resource resourceId="2" type="whole" ></resource>

    • Вставить Ресурс(id = 2) принадлежащий Классу этого Шаблона для скачивания Ресурса
      <resource resourceId="2" type="download" ></resource>

    • Получить ссылку на Ресурс(id = 2) принадлежащий Классу этого Шаблона для самостоятельного конфигурирования.
      <тэг-редактора href="" атрибуты-редактора><resource resourceId="2" type="hyperlink" ></resource></тэг-редактора>
      <тэг-редактора src="" атрибуты-редактора><resource resourceId="2" type="hyperlink" ></resource></тэг-редактора>

Шаблон не может содержать:
  • непарные HTML теги ( например вместо <br> нужно использовать <br></br>)
  • тег <root></root>

Атрибут типа XMEMO

Атрибут типа XMEMO - это XML текст.
Атрибут типа XMEMO может содержать теги Системы для XMEMO:
  • HTML разметку (важно! необходимо использовать всегда парные метки, например <br></br> вместо <br>)
  • теги Системы для XMEMO:
    • Вставить гиперссылку (type = "hyperlink") на Объект (id = 7), связанный Агрегацией (id = 2)
      Он жил в <aggregation aggregationId="2" templateId="5" objectId="7" type="hyperlink">Новосибирске.</aggregation>

    • Вставить Объект (type = "objectInTemplate", id = 7) связанный Агрегацией (id = 2)
      <aggregation aggregationId="2" templateId="5" objectId="7" type="object"></aggregation>

Атрибут типа XMEMO не может содержать:
  • непарные HTML теги ( например вместо <br> нужно использовать <br></br>)
  • тег <root></root>
